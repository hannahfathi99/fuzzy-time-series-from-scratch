# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and inclusive environment for everyone participating in this project.

Participants are expected to interact professionally and respectfully regardless of

* background
* experience
* nationality
* gender
* identity
* level of expertise

## Expected Behavior

Please

* be respectful
* provide constructive feedback
* focus discussions on technical topics
* acknowledge differing viewpoints
* communicate professionally

## Unacceptable Behavior

The following behaviors are not acceptable.

* harassment
* discrimination
* offensive language
* personal attacks
* intimidation
* deliberate disruption of discussions

## Scope

This Code of Conduct applies to

* GitHub Issues
* Pull Requests
* Discussions
* project documentation
* repository communication

## Enforcement

Project maintainers reserve the right to remove comments or contributions that violate these guidelines.

Thank you for helping maintain a positive and collaborative environment.
